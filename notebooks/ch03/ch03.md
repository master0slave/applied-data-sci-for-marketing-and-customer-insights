# Exporatory and Regression Analysis

---
หากการตลาดแบบเดิมคือการรู้ว่าวันนี้ขายดีไหม `Explanatory Analysis` คือการเข้าไปในครัวเพื่อหาคำตอบว่า "ทำไม" ลูกค้าถึงติดใจ (เป็นเพราะรสเค็มของเกลือ หรือความหอมของพริกไทย?) เพื่อที่เราจะได้ปรับสูตรให้ถูกใจลูกค้ามากขึ้นในอนาคต

## 1. Linear Regression: การทำนาย `ปริมาณ` (เหมือนการกะตวงเครื่องปรุง)

ใช้เมื่อเราอยากรู้ "ตัวเลข" หรือ "ปริมาณ" ที่จะเกิดขึ้น เช่น ยอดขายจะเป็นกี่บาท

สมการของ Linear Regression คือ
\[
    y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n + \epsilon
\]

## 2. Logistic Regression: การทำนาย `การตัดสินใจ` (เหมือนการเดาใจว่าลูกค้าจะกินไหม)

ใช้เมื่อผลลัพธ์มีแค่ 2 ทางเลือก เช่น ซื้อหรือไม่ซื้อ, ตอบรับหรือไม่ตอบรับ (Binary Outcome)

สมการของ Logistic Regression คือ
\[
    \text{logit}(p) = \ln\left( \frac{p}{1-p} \right) = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n
\]
